---
name: Flutter Commands
description: Common CLI commands for running, testing, analyzing, and building the app
type: project
---

```bash
# Run the app
flutter run

# Run all tests
flutter test

# Run a single test file
flutter test test/widget_test.dart

# Analyze / lint
flutter analyze

# Install dependencies
flutter pub get

# Build release APK (Android)
flutter build apk --release

# Build release (iOS)
flutter build ios --release

# Regenerate launcher icons after changing assets/images/launcher.png
flutter pub run flutter_launcher_icons
```
